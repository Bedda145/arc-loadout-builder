# ARC Raiders Loadout Builder

A fast, lightweight web tool for building, pricing, and sharing loadouts for *ARC Raiders*. Built to help players easily optimize their in-game credits and weight limits without needing a spreadsheet. 

### Features
* **Real-Time Calculation:** Instantly updates total cost, weight, and ergonomics as you add items.
* **One-Click Sharing:** The URL updates automatically, allowing you to share your exact build instantly.
* **Community Driven:** Free, open-source, and actively maintained alongside the latest game patches.
