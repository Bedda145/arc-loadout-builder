About The Project

Hi, I'm Bedda145. I'm a Year 2 student at LBS Kreativa Gymnasiet in Gothenburg.

I built this tool to help the ARC Raiders community calculate crafting costs and optimize builds. This project helps me learn Front-End Development (HTML, JS, Tailwind) while making something useful for the game we love.
